User Routes
Method Path Description
POST /register Register a new user.
POST /login Login a user that's registered.
GET /users get all registered users
GET /user/id Get a user by using the id.
DELETE /user/:id Delete a product by using it's id.
PUT /user/:id Update an user profile by using the the id.

DELETE /product/:id Delete a single Product by using the id.
